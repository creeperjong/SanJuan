# About cheat code

在`設定`→`金手指` 中，可以輸入特定的作弊碼來對遊戲達到特殊效果

- Libcli

    開場即獲五點vp

- IDidntFinishTheBonus

    除了你以外，每位玩家開場即獲五點vp

- Neokent

    開場直接獲勝

- COVID-19

    為彈性配合政府防疫措施，遊戲將無法進行

- WearAFaceMask

    場上將戴上口罩，將無法檢視
